                                                  Chris Complex 5 years ago

                                                  Thanks!

                                                  Yeah, gathering all the coordinates is a pretty long and painful process at times. A good idea for someone who would want to speed that process up a bit -- also something that I didn't think about while writing -- would be to create a 'values.py' file or something, and then open that in an interactive python session, and have all of the get_pixel() style calls write the coordinates and values it generates into a dictionary stored in the value.py file.

                                                  That way, rather than manually copying everything from the interpreter into your actual code, you could just import the module and have access to all of your coordinates that way.

                                                  Anyway.. glad you liked it!
